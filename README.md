🎯 Fun Media Print

Application web de gestion d’impression permettant de gérer les commandes clients, générer des bons de livraison, créer des QR codes et exporter des documents en PDF.

---

🚀 Aperçu du projet

Fun Media Print est une application web moderne conçue pour :

- Gérer les commandes d’impression
- Suivre les clients
- Générer des bons de livraison professionnels
- Créer automatiquement des QR codes
- Exporter les documents en PDF
- Utiliser une interface simple et mobile-friendly

---

🧰 Technologies utilisées

- HTML5 / CSS3
- JavaScript (Vanilla)
- Bibliothèques externes :
  - QRCode.js (génération QR code)
  - html2canvas (capture écran)
  - jsPDF (export PDF)

---

🔐 Authentification
